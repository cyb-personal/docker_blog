# 🚀 博客论坛项目 - 一键部署教程

## 项目介绍

基于 SpringCloud + Vue + MySQL + Redis 实现的前后端分离博客论坛系统，

**一键部署，无需配置环境，无需导入 SQL！**

## ✨ 项目特性

- 前后端分离架构
- 用户注册 / 登录
- 帖子发布、评论、点赞
- 网关统一入口
- Docker 容器化部署
- **一条命令启动全套服务**

## 🖥️ 部署要求

- 安装 **Docker Desktop** (Windows/Mac/Linux 通用)

## 🔧 一键启动（仅需 2 步）

### 1. 下载配置文件

```
curl -O https://raw.githubusercontent.com/cyb-personal/docker_blog/main/docker-compose.yml
```

### 2. 启动项目

```
docker compose up -d
```

## 🌐 访问地址

- **前端页面**: http://localhost:3000
- **后端网关**: http://localhost:8080

## 📌 服务说明

- MySQL 8.0（自动建库建表，无需 SQL）
- Redis 7.0
- 用户服务
- 帖子服务
- 评论服务
- 点赞服务
- 网关服务
- 前端页面

## ⛔ 停止项目

```
docker compose down
```

## 🔄 彻底重置（清空数据库）

```
docker compose down -v
```
